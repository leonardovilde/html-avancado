CSS Animation — 10 exemplos didáticos

Propriedades: animation-name, duration, timing-function, delay, iteration-count, direction, fill-mode, play-state.

Passe o mouse sobre alguns exemplos para pausar. A página respeita prefers-reduced-motion.

1Fade in/out

Opacidade variando continuamente com ease-in-out.

@keyframes fade + animation: fade 2.2s ease-in-out infinite;

2Movimento horizontal (alternate)

A bolinha se move de um lado ao outro com alternate.

animation-direction: alternate evita “teletransporte”.

3Quicar (bounce)

Deforma levemente ao tocar o chão (squash & stretch).

Cúbica personalizada no timing-function.

4Spinner

Rotação contínua com linear.

animation: spin 1s linear infinite;

5Pulso (pausa no hover)

Interação: animation-play-state: paused no hover.

Demonstra controle da animação via pseudo-interação.

6Máquina de escrever (steps)

Uso de steps(n) para revelar caractere a caractere.

CSS animação é divertido!
animation: typing 3.2s steps(22) infinite alternate;

7Barra de carregamento

De 0% até 100% com linear.

Ideal para simular progresso.

8Skeleton shimmer

Simula conteúdo carregando com background-position.

Muito usado em placeholders de listas/cards.

9Steps(n) em 5 posições

Marcador “salta” por posições discretas (sem suavizar).

animation: jump 2.5s steps(4) infinite; (5 paradas).

10Stagger com fill-mode: both

Itens aparecem em sequência usando animation-delay.

HTML CSS JS React Node
both mantém o estado final após a animação.

ExtraVárias animações juntas

Rodando e pulsando ao mesmo tempo (lista de animações).

animation: spin ..., pulse ...;
Exemplos feitos para uso didático. Sinta-se à vontade para adaptar tempos, timing-functions e direções.
