O que são Media Queries

as media queriespermite que o dev aplique estilos diferentes dependendo das caracteristicas do dispositivo ou tela. O caso mais comum é alterar o layout conforme a largura da tela, para que o site fique responsivo e adptado a celulares, tables e monitores

  • @media(max-qidht: 760px): Aplica estilos para as telas pequenas (smartphones e tablets em modo retraro)
  • @media (min-widht: 1200px): Aplica estilos para telas grandes(monitores full hd,2k e 4k)
  • Também é possivel usar condições como orientation: landscapeou prefers-colors-schema.
