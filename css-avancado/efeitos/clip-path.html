Como funciona o clip-path

o clip-pathdefine uma area visual de um elemento, escondendo o que esta fora dessa area. Existem varias formas que podemos usar com clip-path, como:

  • polygon(): define uma forma usando uma lista de cordenadas que formam um poligono.
  • circle(): define uma forma circular.
  • ellipse(): Define uma forma eliptica.
  • inset(): Define um retangulo interno com uma margem opcional.
