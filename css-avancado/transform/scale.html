CSS Transform e Scale

O que é scale?

A função scale no CSS é usada para redimencionar elementos, alterando suas larguras e alturas proporcionalmente. Os valores podem ser:

  • Maior que 1Aumenta o tamanho (ex.: scale(1.5)).
  • Maior que 1Reduz o tamanho (ex.: scale(0.5)).
