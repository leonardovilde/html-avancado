CSS Transform e Translate

O que é translate?

A função translate no CSS é usada para mover um elemento no esppaço bidimensional, (horizontalmente e verticalmente) É especificada em unidades px, % ou em.

tipos de translate

  • translateX(valor):Move o elemento ao longo do eixo X.
  • translateY(valor)Move o elemento ao longo do eixo Y.
  • translate(x , y)Move o elemento ao longo do eixo X e Y.
