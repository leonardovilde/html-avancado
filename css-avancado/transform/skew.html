CSS Transform e Skew

O que é Skew?

A função Skew no CSS é usada para inclinar ou distorcer um elemento ao longo dos seus eixos X e Y. A inclinação é especificada em graus deg.

tipos de Skew

  • skewX(valor):Inclina o elemento ao longo do eixo X.
  • skewY(valor)Inclina o elemento ao longo do eixo Y.
  • skew(x , y)Inclina o elemento ao longo do eixo X e Y.
