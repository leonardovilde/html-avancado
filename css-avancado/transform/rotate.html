CSS Transform e Rotate

a propriedade transformno css perimite aplicar transformações a um elemento, como redimencionamentos, rotação , inclinação ou movimentação. É uma ferramenta poderosa que pemirte criar efeitos visuais sem a necessidade de scripts

o que é rotate?

a função rotate dentro da propriedade transformé usada para girar um elemento em torno de um ponto de origem. A rotação é especificade em graus deg, mas tambem pode usar outras unidades como radianos radou frações de voltaturn

Exemplos basicos

transform: rotate(45deg) - Este codigo gira o elemento em 45 graus no sentido horario.
